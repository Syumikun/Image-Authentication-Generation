## なんぞや？

Image-Authentication-Generation(IAG)は「securimage」を少し改造し、自分で認証画像の文字を指定できるプログラムです。尚、使い方等は現在作成中です。

※このプログラムはセキュリティ上の理由から、本来の目的(画像認証)での使用はしないでください。
